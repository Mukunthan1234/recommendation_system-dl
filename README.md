ğŸ“˜ Methodology
1. Data Preprocessing

Loaded and cleaned dataset

Encoded users and items into numeric IDs

Sorted interactions by timestamp

Created train, validation, and test splits using a leave-last-two method per user

2. Negative Sampling

Since real-world e-commerce data contains only positive interactions, negative samples were generated by selecting items users did not interact with and labeling them as 0.

3. Neural Collaborative Filtering Model

The NCF model includes:

User Embedding Layer

Item Embedding Layer

Concatenation of embeddings

Multi-Layer Perceptron (128 â†’ 64 â†’ 32 units)

Sigmoid output predicting probability of interaction

4. Evaluation

Binary cross-entropy loss
